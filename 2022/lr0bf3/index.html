canal同步mysql数据到es

发表于2022-07-11|更新于2022-12-03
|阅读量:

环境信息

  • docker
  • mysql 8.0.25
  • canal 1.1.5
  • canal-admin 1.1.5
  • canal-adapter 1.1.5
  • elasticsearch 7.3.0

环境搭建

mysql

修改 my.ini，开启 binlog

使用 docker-compose 启动 mysql

elasticsearch

使用 docker-compose 搭建一个单机版的 es，并且启动 kibana 控制台方便一会操作

kibana.yml

canal-admin

下载 canal-admin 的运行脚本

执行 canal_manager.sql 初始化 canal-admin 的数据表
启动 canal-admin

访问http://localhost:8089/ 密码是 admin/123456

canal

创建一个专用的 canal 账号

下载 canal 启动脚本

运行 canal

canal 启动成功之后登陆 canal-admin 创建一个 example 的 instance 一会会用到
一个 instance 对应一个 mysql 实例

canal-adapter

canal-adapter 官方没有提供镜像，这里我们自己做一个。
Dockerfile

docker-compose.yml

启动 canal-adapter
